### Setup:
Setup a python environment & create a .env file to add openai api key:
```
OPENAI_API_KEY = sk-xxxxxxxxxxx
```

To install requirements:
```
pip install -r requirements.txt
```
To run the application:
```
streamlit run main.py
```
